[gitの初期設定] (作業ディレクトリ内) git init [ステージ上に加える] git add [状態を確認する] git status [コミットをする] git commit [差分を確認する] git diff [変更履歴を確認する] git log [ファイルの削除を記録する] # ファイルごと削除をする git rm <ファイル名> [ファイルの移動を記録する]' git mv <旧ファイル><新ファイル> ローカルの内容をリモートリポジトリに送ること = push [リモートリポジトリを新規追加する] git remote add origin <リモートリポジトリのurl> [リモートリポジトリへ送信する] git push <リモート名> <ブランチ名>
